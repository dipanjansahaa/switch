# AI Text Summarizer

A simple command-line AI Text Summarizer built with Python and Ollama using the Llama 3.2 model.

The application takes a piece of text as input and generates a summary based on the user's preferred length and format.

---

## Features

- Summarize any text using AI
- Choose summary length:
  - Short
  - Medium
  - Long
- Choose summary style:
  - Bullet Points
  - Paragraph
- Runs locally using Ollama
- No OpenAI API credits required

---

## Tech Stack

- Python
- Ollama
- Llama 3.2 (3B)
- dotenv
